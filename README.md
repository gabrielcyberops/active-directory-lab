# Active Directory Lab

## Overview

This project documents the deployment and configuration of a Windows Server 2022 Active Directory lab environment.

The goal of this lab is to simulate a small enterprise infrastructure and gain hands-on experience with:

- Active Directory Domain Services (AD DS)
- DNS configuration
- Group Policy Objects (GPO)
- User and group administration
- Windows enterprise networking
- Security hardening concepts

---

## Lab Environment

| Component | Details |
|---|---|
| Hypervisor | VMware Workstation |
| Server OS | Windows Server 2022 |
| Domain Name | gabriel.local |
| Domain Controller | DC01 |
